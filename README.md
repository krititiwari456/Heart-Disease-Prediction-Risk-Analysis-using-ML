🫀 **Heart Disease Prediction System**
📌 **Overview**

This project is a Machine Learning-based Heart Disease Prediction System that analyzes patient health parameters to predict the risk of heart disease. The system provides a risk percentage, health insights, and visual analytics through an interactive web-based interface.

The goal of this project is to assist in early detection and promote preventive healthcare.

🚀 **Features**
✅ Predict heart disease risk using ML models
📊 Risk percentage with classification (Low / Medium / High)
💚 Health score calculation
📈 Graphical visualization (risk distribution & trends)
🧠 Basic health insights & suggestions
📜 Downloadable report (PDF)
📂 Prediction history tracking
🔐 Multi-user login system

🧠 **Machine Learning Models Used**
Logistic Regression (Final Model ✅)
Random Forest
Support Vector Machine (SVM)
K-Nearest Neighbors (KNN)

📊 **Dataset**
Cleveland Heart Disease Dataset
Source: UCI & Kaggle

**Features include:**

Age, Sex
Chest Pain Type
Blood Pressure
Cholesterol
Heart Rate
Fasting Blood Sugar
ECG Results
And more...

🛠️ **Technologies Used**
Programming Language: Python
Libraries: NumPy, Pandas, Scikit-learn, Matplotlib
Interface: Web-based (Streamlit)
Tools: VS Code

🎯**Results**
Logistic Regression Accuracy: ~88%
Provides accurate prediction with real-time analysis
User-friendly and interactive system
